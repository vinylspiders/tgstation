can_see_target
/datum/bt_node/decorator/can_see_target

Gates child on a blackboard key holding a visible, non-deleted atom. If the target is null, deleted, or not visible to the pawn, clears the key and returns BT_FAILURE.

Vars

keyThe blackboard key whose value is the atom to validate.
rangeVisibility range passed to can_see(). Default matches typical bot search radius.
