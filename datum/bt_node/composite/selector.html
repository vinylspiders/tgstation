/datum/bt_node/composite/selector

Selector node: ticks children in order. Returns the first non-BT_FAILURE result (BT_SUCCESS or BT_RUNNING), stopping further evaluation. Returns BT_FAILURE only if all children fail.

Resumes from the last RUNNING child index rather than restarting from child 1.
