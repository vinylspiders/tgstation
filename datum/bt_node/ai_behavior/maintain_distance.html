maintain_distance
/datum/bt_node/ai_behavior/maintain_distance

Moves away from a target if too close, or toward it if too far, to stay within the blackboard-keyed distance band.

Vars

approach_movement_typeMovement type to use while approaching a target that's too far. Null falls back to the controller's default movement type. Reset on finish.
max_dist_keyBlackboard key holding the maximum desired distance.
min_dist_keyBlackboard key holding the minimum desired distance.
movement_failedSet by on_movement_failed() when the movement system gives up pathing.
target_keyBlackboard key holding the atom to keep distance from.

Procs

retreatSteps one tile away from target using backstep avoidance, falling back to shuffled directions if blocked.
